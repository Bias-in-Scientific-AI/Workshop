# Bias in Scientific AI Workshop Website
Website: [https://bias-in-scientific-ai.github.io/Workshop/](https://bias-in-scientific-ai.github.io/Workshop/) <rb>
Static GitHub Pages site for the Schmidt Sciences Community Initiative Fund workshop:

**Bias in Scientific AI**  
12-15 April 2027  
University of Sao Paulo, Brazil

## Structure

- `index.html` - landing page
- `speakers.html` - invited speakers and panelists
- `program.html` - workshop program with placeholder links for recordings, documentation, and GitHub resources
- `schedule.html` - redirect to `program.html` for old links
- `logistics.html` - venue, travel, accommodation, and local information
- `organizers.html` - organizing team
- `acknowledgement.html` - funder and partner acknowledgements
- `contact.html` - contact details
- `registration.html` - registration information
- `assets/css/styles.css` - shared site styles
- `images/` - hero image and future event images

## GitHub Pages

Enable GitHub Pages from the repository settings and publish from the main branch root. No build step is required.

## Editing

Replace placeholder speaker, organizer, program, contact, and registration text directly in the HTML files. Add photos, venue images, and logos to `images/`.
